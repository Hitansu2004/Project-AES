E1

PCB Defect

Requires Tech

Main printed circuit board error. Communication breakdown between indoor and outdoor units.

lightbulb

Power cycle the unit from the main breaker for 5 minutes. If code returns immediately upon power up, PCB replacement is necessary.

E3

High Pressure

Try Reset First

High pressure switch actuated. Typically caused by poor airflow or overcharged system.

lightbulb

Check outdoor unit for debris (leaves, dirt) blocking the coil. Wash coil gently with a hose and reset unit.

E4

Low Pressure

Requires Tech

Low pressure drop due to potential refrigerant leak or sensor failure.

lightbulb

Do not repeatedly reset. Continuous operation with low refrigerant can permanently damage the compressor.

E7

Fan Motor Err

Requires Tech

Outdoor DC fan motor anomaly. Motor may be locked or experiencing voltage irregularity.

lightbulb

Verify if anything is physically obstructing the outdoor fan blades while the power is off.

F3

Discharge Pipe

Try Reset First

Abnormal discharge pipe temperature. Often related to extreme weather or partial blockage.

lightbulb

Allow unit to rest for 30 minutes. Ensure indoor filters are clean to maintain proper airflow.

H6

No DC Current Detection

Requires Tech

Motor lock or wiring fault. Do not attempt to reset repeatedly.

lightbulb

Motor lock or wiring fault. Do not attempt to reset repeatedly.

P1

Insufficient Voltage

Try Reset First

Voltage supply is unstable. Check power source.

lightbulb

Check if voltage supply is stable. Use a stabilizer if needed.

U4

Communication Error

Try Reset First

Interconnection error between indoor and outdoor units.

lightbulb

Turn off both indoor and outdoor units for 5 minutes, then restart.
